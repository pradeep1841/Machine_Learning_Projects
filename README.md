To add files to git

``````
git add .

``````
OR
``````
git add <file_name>
``````
 
> Note: To ignore the file or folder from git we can write name of file/folder in .gitignore file

To check git status

``````
git status
``````
To check all verion maintained by git

```
git log
```
To create version/commit all changes by git
```
git commit -m "Message"
```
To send version or changes to github
```
git push origin main
```
To check remote url
```
git remote -v
```
